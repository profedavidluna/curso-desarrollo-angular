# Curso de Desarrollo con Angular  

### Información del Curso  
- **Duración:** 32 horas  
- **Nivel:** Intermedio  
- **Versión de Angular:** Angular 19  

### Objetivos Generales  
- Comprender la arquitectura de Angular  
- Crear aplicaciones web utilizando Angular  
- Integrar APIs REST en proyectos de Angular  
- Desarrollar un proyecto integrador final  

### Estructura del Módulo  
| Módulo | Descripción                                        | Duración |  
|--------|--------------------------------------------------|----------|  
| Módulo 1 | Introducción a Angular y TypeScript               | 8h       |  
| Módulo 2 | Componentes, Módulos y Servicios                  | 8h       |  
| Módulo 3 | Enrutamiento y Formularios                         | 8h       |  
| Módulo 4 | Consumo de APIs REST y Proyecto Integrador        | 8h       |  

### Requisitos Previos  
- Conocimientos básicos de JavaScript  
- Familiaridad con HTML y CSS  

### Herramientas Requeridas  
- Node.js  
- npm  
- Angular CLI v19  
- VS Code
